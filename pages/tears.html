TEARS & related

I made a web app that aggregated entry-level remote job listings for software engineers from various job boards. The system automatically updated and posted new listings every 10 to 90 minutes.

I initially self-hosted the platform on a Hetzner VPS server, though I later migrated to AWS Lightsail for more cost-effective hosting in Seoul, South Korea. I deployed the platform using Docker containers with Nginx serving as a reverse proxy. I built the tech stack with Ruby on Rails for the backend, Python for data processing, React for the frontend, and PostgreSQL for database.

In November 2024, I updated the system to use Flask and Jinja2 templates for improved data parsing. When I tried to revisit this project in May 2025, I found out that LinkedIn implemented stronger anti-scraping measures.
