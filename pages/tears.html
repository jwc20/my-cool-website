TEARS & related

This is (or was) a platform that aggregates entry-level remote job listings for software engineers from various job boards. The system automatically updates and posts new job listings every 10 to 90 minutes. The project is self-hosted on a Hetzner VPS server (later I migrated to AWS Lightsail for cheaper server cost in Seoul, South Korea), deployed with Docker, and uses Nginx to reverse proxy IP addresses. The tech stack includes Ruby on Rails for the backend, Python for data processing, React for the front end, and PostgreSQL for the database.

[Update 2024-11-03]
Now using Flask and Jinja2 template for parsing data.

[Update May 2025]
I tried revisiting this project but I became homeless from paying the VPS server fees, and LinkedIn had added more measures to prevent scraping job posts. :(
